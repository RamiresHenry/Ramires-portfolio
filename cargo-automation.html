Cargo Status Automation

📋 Project Overview

This project automated the cargo status update process using Python. Previously, the team spent 4 hours daily manually updating cargo statuses. After automation, the same task takes just 10 minutes.

⚙️ Technologies Used

Python Pandas Openpyxl Sankhya ERP ETL

📊 Key Results

✓ Reduced processing time from 4 hours to 10 minutes (96% reduction)
✓ Eliminated manual data entry errors
✓ Real-time dashboard updates
✓ Increased update frequency from daily to hourly

🔧 How It Works

1. Extract data from Sankhya ERP via API
2. Transform data using Pandas (cleaning, validation)
3. Apply business rules for status classification
4. Load data into tracking dashboard
5. Send automated notifications
