Automated Stock Control System

📋 Project Overview

Excel-based automated system monitoring zero and minimum stock levels for over 20,000 products across 8 sectors at BA Elétrica. Integrated with TOTVS ERP, eliminating manual control inaccuracies.

⚙️ Technologies Used

Excel TOTVS ERP VBA Inventory Control Data Analysis

📊 Key Results

✓ Monitored 20,000+ products in real-time
✓ Eliminated manual inventory errors
✓ Improved demand fulfillment across all 8 sectors
✓ Reduced stockouts by 45%
✓ Automated reordering alerts

🔧 How It Works

1. Extract inventory data from TOTVS ERP
2. Automatically calculate min/max stock levels
3. Flag items below safety stock
4. Generate automatic purchase orders
5. Send email alerts to procurement team
